Authentication

Up 
post /login
Logs in and returns the authentication cookie (loginPost)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body LoginRequest (required)
Body Parameter — A JSON object containing the username and password.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully authenticated -the user is in the system. The session ID is returned in a cookie named <code>JSESSIONID</code>.

401

incorrect username/password Error401

500

Server error occured Error500
Up 
post /register
Sign up, adding new user to the system and return authentication. (registerPost)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body user (required)
Body Parameter — A JSON object containing the register parameters.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully registerd, new user is in the system. The session ID returned a authentication cookie - <code>session</code>.

409

Username taken Error409

500

Server error Error500

Recipes

Up 
get /reciepes/search/query/{searchQuery}/amount/{num}
Search a recipe (reciepesSearchQuerySearchQueryAmountNumGet)
Search recipe by name

Path parameters

searchQuery (required)
Path Parameter — The name of recipe to enter the query.
num (required)
Path Parameter — number of results that will return

Query parameters

cousine (optional)
Query Parameter — Kind of cousine to search by.
diet (optional)
Query Parameter — Kind of diets recipes to search by.
intolerance (optional)
Query Parameter — Kind of diets recipes to search by.

Return type

map[recipe]

Example data

Content-Type: application/json
{
  "157259" : {
    "id" : 157259,

    "image" : "recipe.jpg"

}

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

200

keys and values by the given amount while the key is recipe id and value is object of recipe that contain recipe details.
